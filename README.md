# K8s Cluster with Vagrant & Ansible

## TL;DR
Needs:
1. Vagrant
1. Virtualbox
1. `vagrant plugin install vagrant-vbguest`

## What does it do?
More up to date Version of [https://kubernetes.io/blog/2019/03/15/kubernetes-setup-using-ansible-and-vagrant/](https://kubernetes.io/blog/2019/03/15/kubernetes-setup-using-ansible-and-vagrant/) while using `ansible_local` to support Windows.
